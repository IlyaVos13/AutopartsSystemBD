## Роли пользователей

В системе используются три роли:

### Администратор
- может добавлять пользователей
- может изменять роли пользователей
- может изменять пароли пользователей
- может удалять пользователей
- может очищать тестовые данные
- может выполнять все действия с поставщиками, деталями, закупками и ценами

### Сотрудник отдела закупок
- может добавлять поставщиков
- может добавлять детали
- может назначать детали поставщикам
- может регистрировать закупки
- может добавлять изменения цен
- не может управлять пользователями
- не может очищать все данные

### Руководство
- может просматривать данные
- не может добавлять, изменять или удалять данные
- не может управлять пользователями
