# AutopartsSystemBD

Информационная система учета закупок автозапчастей

Используемые технологии
- C#
- Windows Forms
- PostgreSQL
- Npgsql

Как запустить
1. Создать базу данных PostgreSQL с названием `autoparts_db`
2. Выполнить SQL-запросы из файла `database.sql`
3. Открыть проект в Visual Studio
4. В файле `DataStore.cs` указать свой пароль от PostgreSQL в строке подключения.
5. Запустить проект

Основные функции:
- добавление поставщиков
- добавление деталей
- назначение детали поставщику
- регистрация закупок
- регистрация изменения цены
- просмотр истории цен
- очистка данных
